Hardware I use

When my peers see my stuff they wonder how my keyboard cost more than my laptop. And how I would love an Apple Magic Trackpad while avoiding Apple iCloud.

Laptops

My main laptop is a ThinkPad X230, released in 2012. I bought mine secondhand via a local classified advertisement website, and it included the ThinkPad Ultrabase, which I use at home daily for working at the desk. Old ThinkPads are designed for long term corporate use and lasts forever due to the inner roll cage.

The secundary laptop is a GPD Pocket 2, released in 2018. I received this as a test device for my participation in GPD Linux configuration and driver development project (Shenzhen) and have kept it ever since. This laptop is cute (7” display) until you hook it up on a monitor. This Pocketmonster is great for on the go.

Network Monitor

I’ve setup an IDS (Intrusion Detection System) to monitor network traffic at home. In my attempt to master Network Security, I’ve deployed my IDS at an old Thinkpad (X61). I’m thinking about to replace it for more computing power as the IDS tends to hog the CPU.

Keyboards

How would my keyboard cost more than my laptop, you wonder? While the average “Keyboard and Mouse” setup cost around 20-30 Euros, I tend to spend a fortune on these things. My keyboard at desk is a custom NIZ Micro 82 EC-Ble. It’s an electro capacitive keyboard with custom BKE Redux at 45 grams and custom SA high profile keycaps. I do computing work 12 hours a day on average. As I do, I lay my fingers on these devices and aspire to compute at the highest level of comfort as possible. Or maybe i’m just a geek. Or not maybe, I just am.

I think the sweetspot of keyboard layouts is at 75%. The loss of numpad makes up for space and I find it easier to align my body posture to the computer screen. I use a custom calculator (with MX cherry blue keyswitches) for touchtyping in digits via Bluetooth.

Pointing Devices

Apple Magic Trackpad (2) above any other pointing device, period. After all these years, I still can’t get my hands around a mouse (pun intended). It forces my arm in a very unnatural position as I feel my arm and neck complaining. This is why I find the trackpad awesome: precision and gestures. I’m nowhere slower and the gestures adds a new dimension to desktop and browser control.

I used to use a Logitech MX Anywhere lasermouse for gaming but unfortunately haven’t found time to do desktop gaming lately.

Software I use

These are the programs for my computing needs. The listed programs are used for a while and considered as tried and true, there is no other (my favourites).

Operating system

I use Linux Void, an independent distribution developed entirely by volunteers. Unlike most other distros, Void is not a modification of an existing distribution. It’s package manager and build system have been written from scratch. If you happen to feel like distrohopping and want to try out one without systemd and glibc, than take this green pill. I’ve hopped on this distro since 2018 and never looked back.

Shell

Recently I’ve been using zsh instead of bash. For it’s recursive path expansion, “/u/lo/b” expands to “/usr/local/bin”. It’s basically the bash for modern days with extension plug-ins.

Terminal

URxvt (rxvt-unicode) is a fork of rxvt. I use this because it’s fast, lean and fairly customizable. I’ve spent time on the newer GPU-powered terminals like Alacritty and Hyper but this unique feature of URxvt draws me back everytime: it can run as a daemon to save memory output.

Window Manager (Desktop Environment)

I have no Desktop Manager (KDE/GNOME etc…) but use a Window Manager like i3-gaps instead. This makes me able to control my windows with keyboard only. My windows, wether I want them full-screen or split-screen, are just a few keystrokes away.

My taskbar is based on polybar. It’s shows date and time, battery, volume, wifi, bluetooth, is easy extensible and configurable.

Text editing and programming

I use two text editors: vim and OSS code. I mainly use vim vanilla but for projects I prefer OSS code for it’s plug-in library.

Web Browser

Firefox is a safer standard browser. It blocks cryptomining scripts and social trackers. Chromium as secundary. If it wasn’t for it’s page translation feature, I would have not been using Chromium alongside.

File Manager

I manage my files in the terminal with ranger. Very customizable and efficient, it has vim-like console and hotkeys.

I do use a file manager with GUI for drag and drop stuff: Nemo. Just for it’s looks.

Mail Client

Nowadays rich e-mail is the norm. Mutt takes e-mail back to it’s essentials, no text formatting and no images. It’s good for scripting, to automate sending e-mails. I use this for sending e-mailalerts from my IDS.

RSS Reader

Newsboat. The internet has been blown up by all bells and whistle. It’s better to keep up with the internet through RSS to skip all cookies and ads. Bonus: I browse Youtube channels through RSS and watch videos without having to open a browser.

Spreadsheet

Spreadsheets can be done in the terminal with sc-im. I’m taught to use Excel but all the basic things I do in Excel can be done in sc-im.

PDF Viewer

Zathura opens PDF files instantly. I also use it as a preview window for writing in Markdown. It’s cool to write in Markdown, it’s portable and it’s format is easily converted into a webpage, Word document or yes… a PDF file.

Image modification

GIMP. Although I do all my graphic works on a Mac (Photoshop CS), GIMP comes close. GIMP does mostly everything Photoshop does for my use and with Photoshop keyboard shortcuts in GIMP, the transition is negligible.
