My hardware

My computer keyboard costs more than my computer. Have my expenses for accessoires gone outragious? Or is it just that computing power costs a lot less than before? I don’t think a computer keyboard is just an accessoires, nowadays.

Laptops

My main laptop is a ThinkPad X230, released in 2012. I bought mine secondhand via a local classified advertisement website, and it included the Ultrabase dockingstation, which I use for working at desk. Old ThinkPads are designed for long term corporate use and lasts forever due to the inner roll cage.

My secondary laptop is a GPD Pocket 2, released in 2018. I received this as a test device for my participation in GPD Linux configuration and driver development project (Shenzhen) and have kept it ever since. This laptop is cute (7” display) until you hook it up on a monitor. This Pocketmonster has an i5 M-processor and is great for on the go.

Network Monitor

I’ve setup an IDS (Intrusion Detection System) to monitor network traffic at home. Network Security is a neccesity today. I run my IDS on an old Thinkpad (X61). I’m thinking about to replace it for more computing efficiency as the IDS tends to hog CPU power.

Keyboards

How would my keyboard cost more than my laptop, you wonder? I tend to spend a fortune on these things, ten times higher than average. My keyboard is a custom NIZ Micro 82 EC-Ble. It’s an electro capacitive keyboard with custom BKE Redux at 55 grams and custom SA high profile keycaps. I do computing work 12 hours daily on average. As I do, I lay my fingers on these devices and aspire to compute at the highest level of comfort as possible. Or maybe i’m just a geek. Or not maybe, I just am.

I think the sweetspot of keyboard layout is at 75%. The loss of numpad makes up for deskspace and I find it easier to align my body posture to the computer screen. I use a custom calculator (a numpad device with MX cherry blue keyswitches) for touchtyping in digits via Bluetooth.

Pointing Devices

Apple Magic Trackpad (2) above any other pointing device, period. After all these years, I still can’t get my hands around a mouse. It forces my arm in a very unnatural position as I feel my arm and neck complaining. This is why I find the trackpad awesome: precision and gestures. I’m nowhere slower than with mouse (not honest) and the gestures adds a new dimension to desktop and browser control.

I use a Logitech MX Anywhere lasermouse for gaming but unfortunately haven’t found time to do desktop gaming lately.

My Software

These are the programs I run on the computer. The listed programs are used for a while and considered as tried and true, there are like no others (my favourites).

Operating system

I use Void Linux, an independent distribution developed entirely by volunteers. Unlike most other distros, Void is not a modification of an existing distribution. It’s package manager and build system have been written from scratch. If you happen to feel like distrohopping and want to try out one without systemd and glibc, than take this green pill. I’ve hopped on this distro since 2018 and never looked back.

Shell

Recently I’ve been using zsh instead of bash. For it’s recursive path expansion, “/u/lo/b” expands to “/usr/local/bin”. It’s basically the bash for modern days with extension plug-ins.

Terminal

URxvt (rxvt-unicode) is a fork of rxvt. I use this because it’s fast, lean and fairly customizable. I’ve spent time on the newer GPU-powered terminals like Alacritty and Hyper but this unique feature of URxvt has kept it as my main terminal emulator: it runs as daemon and saves memory output.

Window Manager

I have no Desktop Manager (KDE/GNOME etc…) but use a Window Manager like i3-gaps instead. This makes me able to control my windows with keyboard only. My windows, wether I want them in full-screen or split-screen, are just a few keystrokes away.

My taskbar is based on polybar. It’s shows date and time, battery, volume, wifi, bluetooth, is easy extensible and configurable.

Text Editor

My main tool. “The pen is mightier than the sword”. I use Visual Code (OSS Code with disabled telematry) and vanilla vim for everything else.

Web Browser

Firefox is a safer standard browser. It blocks cryptomining scripts and social trackers. Chromium as secundary for page translation. Browsh for shameless promotion of terminal power and qutebrowser for keyboard driven browsing.

File Manager

I manage my files in the terminal with ranger. Very customizable and efficient, it has vim-like console and hotkeys. From time to time, a file manager with GUI for drag and drop stuff: Nemo for the looks.

Mail Client

Nowadays rich e-mail is the norm. Mutt takes e-mail back to it’s essentials, no text formatting and no images. It’s good for scripting. I use Mutt for sending alerts from my IDS.

RSS Reader

Newsboat. The internet has been blown up by all bells and whistle. It’s better to keep up with the internet through RSS to skip all cookies and ads. Bonus: I browse Youtube channels through RSS and watch videos without having to open a browser.

Spreadsheet

Spreadsheets can be done in the terminal with sc-im. I’m taught to use Excel but all the basic things I do in Excel can be done in sc-im.

Word Processing

Most times at work, i’m forced to adapt to the Microsoft 365 environments and use Microsoft Word for word processing. In any other time I write in Markdown, it’s portable and it’s format is easily converted to HTML, Word document or a PDF File. (Write once, export to many)

Photo Editor

GIMP. Although I do all my graphic works on a Mac (Photoshop CS), GIMP comes close. GIMP does almost everything Adobe CS does and my transition was negligible.
